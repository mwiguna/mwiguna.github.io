Routing
Routing is define what url which you want to use.
You will found the routes file in /app/routes.php

How to configure it? 
$route->url("/", "home", "index");
if you have opened route.php you will see the code like above.
That's define of your route, if you access the route "/", the machine will working on "home" Controller in method "index".

If you want to make other routes, you may add route like the code below : 

$route->url("/", "home", "index"); //your route

//login page, will worked by homeController -> loginPage Method
$route->url("/login", "home", "loginPage");

//admin page, will worked by homeController -> adminPage Method
$route->url("/admin", "home", "adminPage");
So, the term of routes can interpreted like this : 

$route->url("/", "home", "index");
// "/"         => url
// "home" => Controller
// "index" => method
If you don't put the method in the code, the machine will auto describe that route using index method : 
$route->url("/", "home"); // will working on homeController -> index Method
