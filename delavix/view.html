View
View are the files which you want to view to your client.
You may put your file which you want to view in /resource/views/.

There is no structure of the view file, you may make it as you want.

Open file

homepage.php 
<h1>This is my homepage!</h1>
You can view it in controller like this : 
public function homepage(){
	return $this->view('homepage');
}

Show Data

You may also return data : 
public function user(){
	$name = "Rebecca";
	return $this->view('home', $name);
}
And show it in home.php 
echo $name; //output Rebecca

Paginate

You may look at controller part about this method.
and you just need echo in view : 
echo $paginate;
You may use everything you want to view here.
