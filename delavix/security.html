Security
Delavix was made to be secured. We were made it with MVC Pattern and PDO.
But you may increase the security if you want.

Authentification

You may verifying each user with delavix authentification.

Using auth in controller : 

$data = $this->model('DataAdmin')->select()->execute();

if(Auth::user()->id == $data->id_admin){
	//your proccess
}
Using auth in View : 
echo $auth->user()->username;

CSRF Protection

Secure your application from CSRF. You may use token.
First, make the token in your form. 

	<form>
		$auth()->token();
		<input type="tex" name="username">
		<input type="password" name="password">
	</form>
And verify the token in the controller before process : 

	public function insertNew(){
		$this->verifToken($_POST["token"]);
		//your process
	}
If you want to make the static token (in foreach), use this: 

	foreach($data as $user){ 
		
		echo $user->name;
		echo '<a href="delete/'.$user->id.'/'.$auth->staticToken.'">delete</a>';

	}
