Controller
Controller is the main process of this application. Controller will become communicator between routing, model, and view.

You may put your controller in /app/Controllers/
You must writing the name of your controller with camelCase letter, and must be followed 'Controller' in the end of the name.

Example right name of the controller :
homeController.php
userController.php
adminController.php

The Structure of controller

The sturucture of your controller must be similar like this :
class homeController extends Controller {
  //put your method here
}

View your files

You can view your files with method like this : 
class homeController extends Controller {
    //open home.php
  	public function index(){
    	return $this->view('home');
  	}
}

Return Data

You can return datas from your controller into the view : 
class homeController extends Controller {
    //return $user to profile.php
    public function user(){
    	$user = "My name's John";
    	return $this->view('profile', $user);
  	}
}
Show your datas in the view profile.php : 
echo $user; //output : My name's John

Parameter

If you want to use parameter, first you must set it in route. 
$route->url("/user/:param1", "home", "page");
So in controller, you can get it : 
public function user($param1){
	//you may use $param1 now
}

Flash Message

Return the flash message to gives you short messages. 
class homeController extends Controller {
    public function users(){
    	$title = "I wanna check this controller.";
    	return $this->view('users', $title, ["success" => "Success!", "error" => "Error!"]);
  	}
}
There are 2 flash messages here, success and error, you can show it in users.php :
echo $msg->success; //output : Success!
echo $msg->error; //output: Error!
You may modify the flash messages as you want.
